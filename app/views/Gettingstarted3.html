Getting started - Opening and running the kit

If you are now running node at version 20 (or maybe you are close enough) you can now begin to open and run the kit

If you create a new folder on your desktop and call it whatever is relevant to your team etc, but for this example it shall be called tutorials . Next open this folder and place another folder inside that, again I shall use the example of prototype-kit



To open to your prototype, you will need to know a couple of commands for the terminal. To navigate to your prototype first type the following command into the terminal and press enter cd ~/desktop/'first folder name'/'second folder name' so for our example it would be the following cd ~/desktop/tutorials/prototype-kit

Once you can see that you have navigated to the right folder you enter the following command npx govuk-prototype-kit@latest create . This will essentially tell your machine to fill the folder with all of the pieces needed for the prototype kit.

You will then need to type in a few prompts such as npm run dev and reply by typing y and eventually you will be given a link where your prototype is now running locally. HOORAH!
