Getting started - Checking node

Next, we need to check which version of node your machine is running. To do this we first need to open the terminal. If you are using a Mac press command and space, this will bring up a search bar. In the search bar type in 'terminal'.

The terminal can also be found by clicking the magnifing glass in the top right of your screen in the navigation. Again, type 'terminal' and press enter into the search bar and this will open up the terminal.



Above is a video of opening up the terminal and what it looks like



To check the node versioning type node --version into the terminal and press enter. It will display a number, this is the version of node on your machine.

This unfortunately is the part where if the versioning is not relatively up to date, external help is needed. Help to gain temp admin rights to your machine is possible and can be granted by the IT help desk. You can contact them on slack.

GDS reccomend having version 20, which can be downloaded from self service and also can be found on the Node.js website.
